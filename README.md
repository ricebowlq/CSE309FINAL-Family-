# CSE309FINAL (Family+)
An online web application that will allow potential parents to connect to orphanages. Users may choose to adopt children from various orphanages or donate resources to them.

To run this project in local server:

1. Ensure Xampp is installed.
2. Clone the project inside the directory: C:\xampp\htdocs
3. Start Apache and MySQL server from Xampp.
4. In the web browser type: http://localhost/CSE309FINAL-Family-/child-adoptation-home.php
